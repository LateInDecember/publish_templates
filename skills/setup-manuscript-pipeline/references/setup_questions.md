# Phase 0 — 사용자에게 던질 질문

작업 시작 전에 아래를 **한 번에** 물어 답을 받는다. (Cowork에서는 AskUserQuestion 또는 elicitation 폼으로 한 카드에 묶어 묻는 것을 권장.)

## 1. 작업 폴더

- 프로젝트 폴더를 **어디에** 만들까요? (상위 경로의 절대경로)
- 프로젝트 **이름**은? (예: `01_loneliness`)
- 그 폴더가 iCloud/Dropbox 등 클라우드 동기화 폴더인가요? (그렇다면 심볼릭 링크는 사용하지 않음)

## 2. 분석 데이터/결과

- 통계 분석 결과(원고에 들어갈 **표·그림**)가 지금 어디서 생성되나요? 그 폴더의 **절대경로**.
- 아직 분석 전이면 "아직 없음"으로 답하세요. (폴더만 만들어 두고 분석 후 동기화)
- 분석 결과의 "단일 출처"로 쓸 폴더는? (기본: `02_anal/01_R/03_results/06_reporting`)

## 3. 문헌·참고문헌

- 참고문헌을 **Zotero**로 관리하나요? 그렇다면 사용할 **컬렉션 이름**(예: `My Library/<project>/references`).
- 로컬 **PDF 폴더**가 따로 있나요? 있으면 절대경로.
- 기존 `references.bib`가 있나요? 위치는?
- 현재 PDF들이 **이번 논문의 최종 참고문헌**인가요, 아니면 갱신 예정인가요?

## 4. 원고 양식

- 투고할 **저널의 원고 양식/요건**이 있나요? 있으면 파일 또는 핵심 요건(요약 분량, 인용 스타일, 표/그림 규칙, 폰트 등).
- 없으면 이 템플릿의 기본 양식(`manuscript_format/`, 인지 및 생물 심리학회지 2020판)을 사용합니다 — 괜찮나요?

## 5. 분석 도구

- 분석은 **R / Python / MATLAB** 중 무엇인가요? (기본 가정: R)
- (R이면) 렌더에 필요한 패키지를 설치해도 될까요? (`officer, png, stringr, xml2, zip`)
